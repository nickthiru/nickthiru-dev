# Content Directory

This directory contains all content for the nickthiru-dev website.

## Directory Structure

```
content/
├── README.md       # This file
├── posts/          # Published blog posts
└── todo/           # Content ideas and drafts
```

## Overview

### `posts/`

Published blog posts. Each post follows a track-specific template (engineering, business, or product) from the `blog-writer` skill and adheres to the voice policies defined in the `voice-and-style` skill.

### `todo/`

Personal workspace for content ideas and drafts. Files here are excluded from automated AI processing unless explicitly mentioned.
